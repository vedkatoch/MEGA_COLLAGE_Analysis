# MEGA_COLLAGE_Analysis
This project performs data cleaning and analysis on a college dataset.

Steps included:
- Data cleaning using Python (Pandas)
- Handling missing and duplicate data
- Data storage in MySQL database
- SQL query analysis
- Power BI dashboard creation for visualization

Tools Used:
Python, Pandas, MySQL, SQLAlchemy, Power BI

End-to-End Data Analysis Project using Python, MySQL, and Power BI. Data cleaning, transformation, database integration, and interactive dashboard creation using college dataset.
